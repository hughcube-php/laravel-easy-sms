```shell
$ composer require hughcube/laravel-easy-sms -vvv
```

## 安装

```shell
$ composer require hughcube/laravel-easy-sms -vvv

$ php artisan vendor:publish --provider="HughCube\Laravel\EasySms\ServiceProvider"
```
